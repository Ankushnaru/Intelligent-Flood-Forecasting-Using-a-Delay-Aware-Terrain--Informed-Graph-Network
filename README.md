# Intelligent-Flood-Forecasting-Using-a-Delay-Aware-Terrain--Informed-Graph-Network
Intelligent Flood Forecasting using a Delay-Aware Terrain-Informed Graph Network (DT-FPN) models rivers as directed graphs and incorporates terrain-based delays for realistic flood propagation. It improves accuracy over LSTM and ST-GNN in predicting flood levels and peak timing.
